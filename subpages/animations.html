3D animations

During the second semester we were assigned with the specialization project where we were supposed to dive in and extend our knowledge in a specific topic. For this project, I picked 3D animations created in CSS as I found it fascinating, how such objects can be developed only with HTML and CSS.

Throughout the process of creating various animations I learned about CSS properties which enable the 3D appearance of objects. I worked mostly with a transform property which basically makes this idea possible.
